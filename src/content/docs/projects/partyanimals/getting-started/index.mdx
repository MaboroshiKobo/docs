To install the plugin, follow these steps:

<Steps>
1. Download the latest version of the plugin on any of the following platform:
    - [GitHub Releases](https://github.com/MaboroshiKobo/PartyAnimals/releases)
    - [Modrinth](https://modrinth.com/plugin/partyanimals)
    - [Hangar](https://hangar.papermc.io/Maboroshi/PartyAnimals)

2. Place the downloaded `.jar` file into your server's `plugins` directory.

3. Restart your server to load the plugin.

4. (Optional) Install [PlaceholderAPI](https://github.com/PlaceholderAPI/PlaceholderAPI/releases) to utilize built-in placeholders.

5. (Optional) Install [NuVotifier](https://github.com/NuVotifier/NuVotifier/releases) to enable voting features.
</Steps>
