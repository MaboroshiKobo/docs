---
title: Commands & Permissions
description: Detailed list of commands and permissions for PartyAnimals.
---

| Command | Permission | Description |
|---------|------------|-------------|
| `/partyanimals` (or `/pa`) | `partyanimals.use` | Main command to access Party Animals features. |
| `/partyanimals help` | `partyanimals.help` | Displays help information for Party Animals commands. |
| `/partyanimals reload` | `partyanimals.reload` | Reloads the Party Animals configuration without restarting the server. |
| `/partyanimals pinata start <pinata> <location>` | `partyanimals.pinata.start` | Start pinata countdown. |
| `/partyanimals pinata spawn <pinata> <location>` | `partyanimals.pinata.spawn` | Spawn pinata immediately. |
| `/partyanimals pinata killall` | `partyanimals.pinata.killall` | Remove all active pinatas. |
| `/partyanimals pinata spawnpoint add <name>` | `partyanimals.pinata.spawnpoint.add` | Save current location. |
| `/partyanimals pinata spawnpoint remove <name>` | `partyanimals.pinata.spawnpoint.remove` | Remove saved location. |
| `/partyanimals vote check <player>` | `partyanimals.vote.check` | Check player votes. |
| `/partyanimals vote add <player> <amount>` | `partyanimals.vote.add` | Add votes to a player's balance. |
| `/partyanimals vote remove <player> <amount>` | `partyanimals.vote.remove` | Remove votes from a player's balance. |
| `/partyanimals vote set <player> <amount>` | `partyanimals.vote.set` | Set a player's vote balance. |
| `/partyanimals vote send <player> [service]` | `partyanimals.vote.send` | Simulate a vote. |
| `/partyanimals vote migrate <plugin>` | `partyanimals.vote.migrate` | Migrate data from another plugin (e.g., `PinataParty`). |
