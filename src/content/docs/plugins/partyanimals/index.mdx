**Party Animals** is a unique **event plugin** created to elevate server gatherings with interactive, **loot-filled Pinatas**!

This plugin is ideal for servers looking to add excitement to celebrations and reward players, with features that improve the elements of your **server events**.

### Features

**Party Animals** is packed with features designed to bring joy to your community. Check out some of its features:
* **Create and customize** unique Pinata templates.
* **Spawn Pinatas** immediately or start a build-up countdown.
* **Accurate loot distribution** based on hit participation or last hit.
* **Save locations** as **spawn points** to easily respawn Pinatas at specific event spots.
* Have access to **admin tools** to manage active Pinatas.
* **Rich text support** using MiniMessage for all messages.
* Complete and total **configuration control**; decide exactly what drops and when.
* **Performance friendly** entity handling to ensure smooth events.
