Below is an example of a configuration file for the Party Animals plugin.

### `config.yml`

```yml
# Enable debug mode to see detailed logs in the console.
debug: false
# Control which parts of the plugin should be active.
modules:
  # Toggle the pinata system.
  pinata:
    # Enable or disable the pinata system.
    enabled: true
    # Defined spawn points.
    spawnPoints:
      default:
        world: world
        x: 0.0
        y: 100.0
        z: 0.0
        yaw: 0.0
        pitch: 0.0
  # Toggle the voting system.
  vote: false
```
